Sobre o desenvolvimento da página

Na área de desenvolvimento de jogos, a maioria são muito complexos, desenvolvidos a partir de “Engine” que são difíceis de se utilizar e entender, e aqui no Brasil, muito pouco comentados sobre. Nós desejamos monstrar à partir do desenvolvimento de jogos feitos de forma mais simples e com divulgação aberta de seus códigos, uma forma para que outras pessoas possam aprender também.

Nossa proposta é esta, um site de jogo de luta (de código aberto), feito com os conhecimentos obtidos das disciplinas de “Desenvolvimento Web I” e “Desenvolvimento Web II”, desenvolvido a partir de linguagens e tecnologias não tão complexas quanto as de “Engine”, como: NPM, Express, Typescript, Javascript, HTML, CSS, BootStrap, HandleBars e RENDER.
